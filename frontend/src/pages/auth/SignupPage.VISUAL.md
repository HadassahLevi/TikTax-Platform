### Mobile (< 640px)

- Full-width inputs
- Stacked buttons

- Compact progress indicator

### Tablet (640px - 1024px)
- Same as mobile, slightly more padding
- Card max-width: 448px
- Centered on screen

### Desktop (> 1024px)
- Same as tablet
- More prominent shadows
- Hover effects enabled
