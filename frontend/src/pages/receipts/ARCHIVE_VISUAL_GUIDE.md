## ğŸ¯ Click Targets

### Primary Actions
- **Receipt Card**: Navigate to `/receipts/{id}`
- **Add Receipt Button**: Navigate to `/receipts/new`
- **Export Button**: Navigate to `/export`

### Secondary Actions
- **Filter Button**: Open filter modal
- **Sort Button**: Open sort modal
- **View Toggle**: Switch grid/list
- **Filter Chip X**: Remove that filter
- **Clear All**: Remove all filters

### Modal Actions
- **Apply Filters**: Close modal + apply
- **Clear All Filters**: Reset + close modal
- **Sort Option**: Select + close modal
- **Modal X**: Close without changes
